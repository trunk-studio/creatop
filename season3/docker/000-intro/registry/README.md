# Registry

註冊伺服器（registry server）是提供 Docker Repository 服務的主機，

* Docker Hub

  Docker 官方提供的註冊伺服器。

* DockerPool

  中國大陸的 Docker 社群建立的註冊伺服器。

當然，不是所有的 Docker Image 都適合發佈到公開的註冊伺服器，因此也可以建立私有、具存取權限控管的 Registry，例如透過雲端運算服務：

* Amazon EC2 Container Registry

或是在私有伺服器主機架設。
